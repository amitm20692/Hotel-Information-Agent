# Hotel-Information-Agent
An AI-powered agent that answers guest questions for a hotel.

## Features
- Answering questions about Amenities
- Answering questions about restaurant hours
- Answering questions about checkout times

## Architecture and Technologies
- Azure AI foundry
- AI models: gpt-5-mini, gpt-4.1
- Instructions: You are a helpful hotel concierge attending to gues inquiries and concerns. Be nice and have a great attitude towards the customer.
